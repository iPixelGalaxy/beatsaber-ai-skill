# Beat Saber Mod-Specific Skill Files

This directory is intentionally opt-in.

Only read a mod file when the user names that mod, asks to process that mod, or asks to update mod-specific `$beatsaber` knowledge.

## Files

- `BSML.md`: BSML / BeatSaberMarkupLanguage parser, tags, API surface, and migration notes for 1.29.1, 1.40.8, and Latest/1.44.0.
- `BS_Utils.md`: BS_Utils events, menu level selection event, scene lifecycle hooks, and LevelData notes for local 1.44.0.
- `BeatSaberCinema.md`: selected-level Harmony patch pattern, reset patches, and direct TMP UI refresh pattern from local Cinema.
- `MappingExtensions.md`: detailed 1.29.1 to 1.40.8 behavior and 1.44.0 update risks for MappingExtensions.
- `Reactive.BeatSaber.Components.md`: local Reactive.BeatSaber.Components API notes, common components, and GameplaySetup integration caveat.
